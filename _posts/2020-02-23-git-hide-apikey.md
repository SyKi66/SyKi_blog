# 1. 프로젝트의 루트폴더에 `.env` 파일 생성

루트폴더에 `.env` 파일 생성후, 들어가서 변수명 앞에 반드시 `REACT_APP_` 붙여서 변수명을 설정하고 API 키를 입력한다.

```
REACT_APP_API_KEY=AbCdEfG
```
